1. 概要
本ソフトウェア「PDF Con」は、Windows 用のフリーウェアとして
開発された PDF → JPEG 変換ツールです。ドラッグ＆ドロップで
複数ファイルを一括処理でき、変換後の解像度もスライダーで
簡単に調整できます。


2. 動作環境
- Windows 10 以上（x64）
- .NET 9 ランタイム（自己包含版を同梱）
- CPU：Intel i5 相当以上推奨
- メモリ：4GB 以上


3. インストール方法
Setup.exe を実行してください。



4. 使い方

1. アプリを起動します。
2. 画面中央に PDF ファイルをドラッグ＆ドロップします。
3. スライダーで解像度（300～2000 DPI）を設定します。
4. 「変換開始」ボタンを押すと、同じフォルダ内に JPEG が出力されます。


5. 主な機能

- 複数 PDF の一括変換
- DPI スライダー（300～2000、100 刻み）
